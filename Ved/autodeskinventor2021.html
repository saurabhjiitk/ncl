How to install Autodesk Inventor 2021


Run set up file of Autodesk Inventor 2021 "setup.exe"




Click on "Install" button




Select "I accept" button and next




Select "Autodesk Inventor Professional 2021" and next




Select Folder path in local drive.(Choose default) and Install




Wait and watch for few time




When installation is finished, click "Finish" button




Click "Yes" to restart button




Launch Autodesk Inverter 2021, Select "Use a network license"




"Specify a license server" Select Single Licence Server model and fill Server IP "172.31.1.12", Done




you can check Hardware and License details,
