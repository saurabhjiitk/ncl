FAQ-on-Procedure-to-install-Stata12

Here the procedure explains how to install StataSE 12 in windows operating system.



  1. Firstly your have the state setup file.




  1. Run the setup file of stata 12MSI




  1. In welcome to the Stata 12 Installation Wizard Click on Next.




  1. In License Agreement you mus agree with the license agreement below to proceed. So Select "I accept the license agreement" then click on Next.




  1. User Information Enter the 'Full Name' and 'Organization' select Anyone who uses this computer after Anyone who uses this computer after that click on 'next'.




  1. Select Executable Flavor 64-bit SE then Click on 'Next'.





  1. Select Destination Folder then click on 'Next'




  1. Select Default Working Directory and Then Click on 'Next'




  1. Then Ready to Install the Application and click on 'Next.




  1. Here the feaurtes you selected are currently being installed and updation system.




  1. Successfully Stata 12 has been installed then click Finish.
