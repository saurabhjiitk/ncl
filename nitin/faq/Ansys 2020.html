Installation of ANSYS 2020 in Microsoft Windows


Note: Before installation of ANSYS 2020 it is recommended to install "Microsoft Visual Studio Express 2015".We can download both "Microsoft Visual Studio Express 2015" and ANSYS 2020 setup from CC FTP site.

Step 1:After downloading extract the zipped setup file to computer Local Drive. Open the extracted folder and double click on the ANSYS setup.exe file to install software to your computer.


Step 2:Click on "Install ANSYS Products" button as shown in below fig.



Step 3:Now select "I Agree" and click on "Next" button.



Step 4:Click on the "Next" button.



Step 5:Click on the "Next" button.



Step 6:Choose "Yes, automatically configure my interfaces" and then click on the "Next" button.



Step 7:Click on the "Next" button.



Step 8:Click on the "Next" button.



Step 9:Wait for 10 - 15 minutes to complete the installation process and then click on "Next" button.



Step 10:Uncheck "Lounch survey upon exiting" and then click on "Exit" button.


Step 11:Type "ClientANSLIC_ADMIN Utility 2020 R1" in start menu search bar and Run the application as Administrator, now click on " Specify the License Server Machine".



Step 12:Click on "Add Server Machine Specification".



Step 13:Input ANSYS license server name or server IP ("akash3.cc.iitk.ac.in or 172.31.1.198") in "Hostname 1:" as shown in below fig. and then click on "OK" button.
