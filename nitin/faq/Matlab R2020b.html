Installation of Matlab R2020b in Microsoft Windows

Step 1:Download the setup file of Matlab R2020b, after downloading extract the zipped setup file to computer Local Drive. Open the extracted folder and double click on the Matlab R2020b setup.exe file to install Matlab R2020b to your computer.



Step 2:Click on Advanced Options on right hand side of screen and select “I have a File Installation Key” as shown in fig.



Step 3: Click on “Yes” to accept the license agreement and hit “Next” button.



Step 4: Type the file installation key and hit “Next” button.



Step 5: Provide the network license file path by clicking on “Browse” button and then hit “Next” button.



Step 6: Click “Next” button.



Step 7: Click “Next” button.



Step 8: Click on “Begin Install” button.



Step 9: Wait for 10 - 15 minutes to complete the installation process.



Step 10: Click “Close” button to exit the Matlab install process after completion.
