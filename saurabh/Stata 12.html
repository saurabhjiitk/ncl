Stata 12

1. Firstly you have the state setup file, Run the setup file of stata12MSI

2. Click on "Next" to installation of Stata 12

3. Select "I accept the license agreement" and click on "Next".

4. Enter the 'Full Name' and 'Organization' , select Anyone who uses this computer after that click on 'Next'.

5. Select Executable Flavor 64-bit SE then Click on 'Next'.

6. Select Destination Folder then Click on 'Next'

7. Select Default Working Directory and Then Click on 'Next'.

8. Then Ready to Install the Application and Click on 'Next'.

9. Updating component registration.

10. Successfully Stata 12 has been installed then Click Finish.
