Ansys 2020


Note: Install Microsoft Visual Studio Express 2015 before installation Ansys 2020 (if Visual Studio already Installed then we Continue Install Ansys 2020). The Ansys 2020 software is compressed to a single zip file and available for download CC ftp site. Extract the zipped file to local drive and run the setup.


1. Click on the Install ANSYS Products button.






2. Select I Agree and Click on the Next button.






3. Click on the Next button.






4. Enter Ansys License Server IP Add in Hostname. (Network License Server for IIT Campus is akash3.cc.iitk.ac.in) and Click on OK Button.






5. Select the required products (Default Recommended) and Click on the Next button.






6. Select Yes automatically configure my interfaces and click on the Next button.






7. Click on the Next button.






8. Click on the Next button.






9. Wait for several minutes Ansys 2020 installation to complete and click on the Next button.






10. Uncheck Launch survey upon exiting and click Exit button.






11. Open Client ANSLIC_ADMIN Utility from the START menu and click on Specify the License Server Machine.






12. Show ANSYS Licensing Interconnected and ANSYS FLEXIm server machine Specified.
