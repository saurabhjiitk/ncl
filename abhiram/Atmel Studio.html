Installation Steps of Atmel Studio in Micro Soft Window 10

1. Run the setup. Select "I Agree to the license terms and conditions" and Click on the "Next" button.

2. Select the Necessary Architectures you would like work with. Click on "Next" button.

3. Select "Extensions" if you need example projects to start with. Click on "Next" button.

4. Select all "System Validation" option and Click on the "Next" button.

5. Click on the "Install button.

6. Installation will take some time according to system performance

7. If Microsoft Visual Studio 2015 already Installed in Computer then automatically applying Microsoft Visual Studio 2015 Shell (isolated) package. Take a few minute.

8. Installation Complete and Tick the Check Box "Launch Atmel Studio". Click on the "Close" button.
