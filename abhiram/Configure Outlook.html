How to Configure Microsoft Outlook Mail Client for IITK Webmail

1. Click on "Start "and Type "Control" to open "Control Panel" Option.

2. Click on "Mail" Option.

<

3. Click "Add" to add a profile name.

4. Enter any profile name (e.g. "IITK Webmail").and Click on "OK" button.

5. Select "Manually configure server setting or additional server type" option and click on "Next" button.

6. Select " Internet E-mail (to connect to POP or IMAP server )" and click on "Next" button.

7. Fill user and server information as shown below and click on "More Setting".

Incoming Server for External Network -qasid.iitk.ac.in

Outgoing Server for External Network- mmtp.iitk.ac.in

Incoming Server for Internal Network- newmailhost.cc.iitk.ac.in

Outgoing Server for Internal Network- smtp.cc.iitk.ac.in

And port no Incoming server is - SSL\TLS 993

Port no Outgoing Server is -SSL\TSL 465

Note: use "External Network" setting for Laptops/Mobiles/ tablets and "Internal Network" setting only for Desktop.

8. Select "Outgoing Server" tab and Enable "My outgoing server SMTP required authentication.

9. Select "Advance" tab and update the information as shown as below.

10. Click "Close" after Successfully Configuration.

11. Account successfully added and Click on "Finish" Button.

12. Preview for Configure Outlook as below.
