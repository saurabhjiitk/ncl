Installation of Codeblocks 17 in Microsoft Window

1. Run the Codeblocks 17 Setup file “Codeblocks17.12-setup .exe”. Click on the “Next” button.

2. Accept the “License Agreement” and click “I Agree” button.

3. In the next window, Code Blocks will ask you to select components you want to install. Select Full Installation. Click on “Next” button .

4. Select the Destination Folder (keep it default) and Click “Next” button.

5. The Installer will proceed to complete the Installation.

6. Ask the Question “Do you want to run code:: blocks now”. Click “Yes” button.

7. Installation has been completed. Click on the “Next” button.

8. Click on the “Finish” button.
